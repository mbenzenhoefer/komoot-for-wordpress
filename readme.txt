== Description ==
Mit komoot for Wordpress kannst du Wanderungen, Mountainbike- oder Fahrradtouren ganz einfach  als Karte, &Uuml;bersichtskarte oder Liste in deinen Blog einbinden.

Das geht mit jeder Tour, die du auf www.komoot.de geplant hast, mit der komoot iPhone oder Android App aufgezeichnet hast oder als GPX-Datei auf Mein komoot hochgeladen hast.

 * Einbinden einer &Uuml;bersichtskarte aller deiner Touren `[ www.komoot.de/c/BENUTZERNAME/ ]`
 * Einbinden einer einzelnen Tour als Karte `[ www.komoot.de/c/BENUTZERNAME/ALBUM/TOUR/ID ]`
 * Einbinden der Tourliste als Widget in der Seitenleiste deines Blogs


== Installation ==
1. Installiere und aktiviere **komoot for Wordpress** auf deiner Plugin Seite
1. W&auml;hle eine Tour auf [www.komoot.de] (http://www.komoot.de/ "komoot - Wanderungen, Fahrrad und Mountainbike Touren) aus
1. F&uuml;ge den Link zur Tour als Shortcode ein`[ www.komoot.de/c/BENUTZERNAME/ALBUM/TOUR/ID ]`

Wenn du deine Tourliste als Widget in deinen Blog einbinden m&ouml;chtst:
1. Aktiviere das **komoot Tourlisten Widget** auf der Plugin Seite
1. F&uuml;ge das Widget unter Design -> Widgets an der gew&uuml;nschten Stelle ein
