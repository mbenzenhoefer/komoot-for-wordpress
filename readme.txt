== Description ==
Mit komoot for Wordpress kannst du Wanderungen, Mountainbike- oder Fahrradtouren ganz einfach  als Karte in deinen Blog einbinden.

Das geht mit jeder Tour, die du auf www.komoot.de geplant hast, mit der komoot iPhone oder Android App aufgezeichnet hast oder als GPX-Datei auf Mein komoot hochgeladen hast.

 * Einbinden einer &Uuml;bersichtskarte aller deiner Touren `[ www.komoot.de/user/ID ]`
 * Einbinden einer einzelnen Tour als Karte `[ www.komoot.de/tour/TOURNAME/ID ]`


== Installation ==
1. Installiere und aktiviere **komoot for Wordpress** auf deiner Plugin Seite
1. Wähle eine Tour auf [www.komoot.de] (http://www.komoot.de/) aus
1. Füge den Link zur Tour als Shortcode in deinen Post ein`[ www.komoot.de/tour/TOURNAME/ID ]`
