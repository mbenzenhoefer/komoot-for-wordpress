== Description ==
Mit komoot for Wordpress kannst du Wanderungen, Mountainbike- oder Fahrradtouren ganz einfach  als Karte, &Uuml;bersichtskarte oder Liste in deinen Blog einbinden.

Das geht mit jeder Tour, die du auf www.komoot.de geplant hast, mit der komoot iPhone oder Android App aufgezeichnet hast oder als GPX-Datei auf Mein komoot hochgeladen hast.

 * Einbinden einer Übersichtskarte aller deiner Touren `[ www.komoot.de/c/BENUTZERNAME/ ]`
 * Einbinden einer einzelnen Tour als Karte `[ www.komoot.de/c/BENUTZERNAME/ALBUM/TOUR/ID ]`
 * Einbinden der Tourliste als Widget in der Seitenleiste deines Blogs
